# 数据产品经理作品集

## 项目简介

本项目是一个完整的数据产品经理作品集，以视频剪辑软件行业为背景，展示数据产品经理的核心能力。

## 核心能力展示

- 行业调研能力：深入分析剪映、Filmora等产品
- 指标体系设计：建立完整的数据指标框架
- PRD撰写能力：输出专业的产品需求文档
- 数据分析能力：使用Python进行用户评论分析
- 数据可视化：使用Streamlit搭建交互式数据看板

## 项目结构

- 01_Research：行业与产品调研
- 02_Metrics：数据指标体系
- 03_PRD：产品需求文档
- 04_Demo：数据分析与看板

## 核心产出

### 1. 产品调研报告
对比分析剪映与Filmora两款产品的定位、功能、商业模式差异。

### 2. 数据指标体系
为视频剪辑软件设计完整指标框架，包含北极星指标、核心过程指标、诊断分析指标。

### 3. PRD文档
数据管理平台产品需求文档，包含用户分析、功能设计、原型说明、项目计划。

### 4. 数据分析
用户评论分析，包含评分分布、关键词提取、趋势分析。

### 5. 数据看板
交互式数据看板，支持多维度筛选、趋势图表、智能预警、数据导出。

## 技术栈

- Python
- Pandas
- Plotly
- Streamlit

## 如何运行看板

cd 04_Demo/dashboard
streamlit run app.py
